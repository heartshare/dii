为移动而生

Dii 是国内 Discuz 程序的拓展功能，继承Discuz核心的函数，去除了其他累赘的东西，适用于Disxuz二次开发者中使用。

组件丰富，模块化

Dii 结合 amazeui 含近 20 个 CSS 组件、10 个 JS 组件，更有 17 款包含近 60 个主题的 Widgets，可快速构建界面出色、体验优秀的跨屏页面，大幅度提升你的开发效率。

清晰的MVC模式

基于国外的Yii PHP框架，Dii 使用Yii简单的MVC模式进行二次开发，今后将继续把Yii强大的功能转移过来，实现与Discuz功能的无缝结合，可以实现Discuz插件的转移与独立目录运行支持，为您节省大量的开发时间。

轻量级，高性能

Dii 非常注重性能，基于轻量的 Discuz内核 开发，并使用 amazeui 来做动画交互，平滑、高效，更适合移动设备，让你的 Web 应用可以高速载入。
